Known limitations — what this tool can't track
  • Claude Design. Anthropic doesn't release token counts for it — there's nothing to read, on or off your machine.
  • claude.ai chat & co-work (the web app). That runs entirely on Anthropic's servers; nothing is ever written to your machine for it.
  • Reasoning effort & fast mode. Not recorded in the local transcript format Claude Code writes, so there's no field to read.
Only Claude Code / Claude CLI usage is tracked (local transcripts under %USERPROFILE%\.claude\projects\) — including a local model like Ollama if you ever point Claude Code at one (shown as "Local", always $0).
